# Sift-PHP 
Sift-PHP try to imitate the [sift.js](https://github.com/crcn/sift.js) node library

### Supported operations
For this version the only supported operation are:
* $and
* $or
* $nor
* $not
* $eq
* $ne
* $gt
* $gte
* $lt
* $lte
* $in
* $nin
* $size
* $exists
* $regex
